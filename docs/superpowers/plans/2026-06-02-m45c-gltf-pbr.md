# M45c — Finish glTF PBR Materials Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extract glTF occlusion + emissive textures and the PBR scalar factors, extend the engine `Material`/shader with an emissive map + base-color factor, and wire gltf-viewer + sandbox so glTF models render their authored PBR materials (the DamagedHelmet's metallic visor, AO, glowing emissive, albedo tint).

**Architecture:** `GltfLoader` already resolves albedo/normal/metallic-roughness paths; add occlusion + emissive paths and the metallic/roughness/baseColor/emissive factors to `GltfModel`. `Material` gains `emissiveMap` (sRGB) + `baseColorFactor`; the existing `emissive` Vec3 becomes the emissive factor (× map). Descriptor layout grows binding 8 = emissive (lit 8→9 bindings; skinned bones UBO moves 8→9). `LitUbo` appends `baseColorFactor` (960 bytes). Shader: `albedo *= baseColorFactor`, `emissive = emissive × emissiveMap`. IBL is M46 (out of scope).

**Tech Stack:** Vulkan, tinygltf, runtime glslang, C++23, MSVC, CTest. Build `build-vk`. Spec: `docs/superpowers/specs/2026-06-02-m45c-gltf-pbr-design.md`. Branch: `feat/m45c-gltf-pbr`.

**Build (all tasks):** `cmake --build build-vk --config Debug`
**Tests (all tasks):** `ctest --test-dir build-vk -C Debug --output-on-failure`
