## Summary

Gizmo polish + planar move handles for the editor (M32 + M33).

- **Always-on-top** rendering — a depth-disabled overlay line path (`Renderer::drawLineOverlay` + a second `VkDebugLines` pipeline); the gizmo is never occluded by geometry.
- **Hover highlights + click-grab** — handles brighten on hover; clicking a highlighted handle grabs it, so clicking a handle never deselects the object (the M31 bug).
- **Pivot origin** — the gizmo sits at the entity's transform pivot (rotation-stable). A bounds-center origin drifted for off-pivot / cabled meshes; the pivot fixes that.
- **Smooth drag** — holds the last axis param on degenerate/grazing solves instead of jumping (fixes the M31 twitch).
- **Planar move handles (M33)** — three world-plane quads (XY/YZ/XZ) near the gizmo center in Translate mode; drag one to move the object in that plane (two axes at once) via ray-vs-plane.
